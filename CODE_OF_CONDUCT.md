# Code of Conduct

## Our Pledge

We commit to keeping this project open, respectful, and practical. Contributors, users, and maintainers are expected to treat each other with professionalism and restraint.

## Expected Behavior

- Be direct without being hostile.
- Critique ideas and code, not people.
- Keep discussions focused on the project.
- Assume good intent unless there is clear evidence otherwise.
- Report bugs, regressions, and risks clearly.

## Unacceptable Behavior

- Harassment, slurs, threats, or personal attacks
- Deliberate misinformation
- Spamming, trolling, or disruptive behavior
- Repeated bad-faith arguments after a decision has been made
- Any behavior that makes collaboration materially harder for others

## Enforcement

Maintainers may remove comments, close issues, reject contributions, or restrict participation when behavior is disruptive or abusive.

## Scope

This code of conduct applies to:

- repository issues and pull requests
- wiki pages and documentation
- package submissions and reviews
- project discussion in any official project channel

## Reporting

If you see behavior that violates this code of conduct, open a private issue or contact the maintainers through the project's normal support channel.
